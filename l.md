# ☁️ Introduction to AWS

## What is AWS?

Amazon Web Services (AWS) is a cloud computing platform provided by Amazon that offers on-demand IT resources such as servers, storage, databases, networking, security, and many other services over the internet.

Instead of buying and maintaining physical hardware, users can use AWS services whenever needed and pay only for what they use.

---

# 🚀 Why AWS is Popular

- Scalable infrastructure
- High availability and reliability
- Pay-as-you-go pricing
- Secure cloud environment
- Global data centers (Regions & Availability Zones)
- Supports DevOps, AI, ML, Big Data, and Web Applications

---

# 🌍 Real-World Uses of AWS

Companies use AWS for:
- Hosting websites and applications
- Data storage and backup
- Running virtual servers
- CI/CD and DevOps automation
- Machine learning and analytics
- Kubernetes and container deployment

---

# 🔑 Core AWS Services

| Service | Purpose |
|---|---|
| EC2 | Virtual servers in the cloud |
| S3 | File and object storage |
| VPC | Private network in AWS |
| IAM | User access and permissions |
| RDS | Managed databases |
| CloudWatch | Monitoring and logging |
| Lambda | Run code without servers |
| EKS | Managed Kubernetes cluster |

---

# 🏗️ AWS Global Infrastructure

AWS infrastructure is divided into:

## 🌎 Regions

Different geographic locations around the world.

### Examples:
- Mumbai (`ap-south-1`)
- Frankfurt (`eu-central-1`)
- North Virginia (`us-east-1`)

---

## 🏢 Availability Zones (AZs)

Separate data centers inside a region for high availability.

### Example:
```text
eu-north-1a
eu-north-1b
eu-north-1c
```

---

# 💰 AWS Pricing Model

AWS follows:

```text
Pay-As-You-Go
```

You pay only for:
- Compute time
- Storage used
- Network usage
- Services consumed
